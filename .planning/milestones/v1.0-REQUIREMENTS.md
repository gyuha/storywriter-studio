# Requirements Archive: v1.0 StoryWriter Studio MVP

**Archived:** 2026-05-26
**Milestone:** v1.0 — StoryWriter Studio MVP
**Core Value:** 작가가 챕터를 편집할 때 캐릭터·장소·설정이 자동으로 AI 컨텍스트에 포함되어, 세계관과 일관된 글을 AI가 생성한다.

---

## v1 Requirements (All Complete)

### AUTH — 인증 및 기반 인프라

- [x] **AUTH-01**: 사용자가 이메일/비밀번호로 로그인하면 프론트엔드가 실제 FastAPI 백엔드 JWT를 발급받아 인증 상태를 유지한다
- [x] **AUTH-02**: 사용자가 이메일/비밀번호로 회원가입하면 계정이 생성되고 세션이 유지된다
- [x] **AUTH-03**: 인증된 사용자만 소설 프로젝트 및 콘텐츠에 접근 가능하다
- [x] **AUTH-04**: 관리자가 관리자 페이지에서 사용자 목록을 조회하고 계정을 관리할 수 있다

### PROJ — 소설 프로젝트 관리

- [x] **PROJ-01**: 사용자가 소설 프로젝트를 생성할 수 있다
- [x] **PROJ-02**: 사용자가 자신의 소설 프로젝트 목록을 조회할 수 있다
- [x] **PROJ-03**: 사용자가 소설 프로젝트의 기본 정보를 수정할 수 있다
- [x] **PROJ-04**: 사용자가 소설 프로젝트를 삭제할 수 있다

### CHAP — 챕터 관리 및 에디터

- [x] **CHAP-01**: 사용자가 소설 내에 챕터를 생성할 수 있다
- [x] **CHAP-02**: 사용자가 TipTap 리치 텍스트 에디터로 챕터 본문을 작성할 수 있다
- [x] **CHAP-03**: 챕터 본문이 작성 중 자동으로 저장된다 (debounce 방식)
- [x] **CHAP-04**: 에디터에 현재 챕터의 글자 수가 실시간으로 표시되며 5,000자 목표 달성 시 시각적 표시
- [x] **CHAP-05**: 사용자가 드래그앤드롭으로 챕터 순서를 재정렬할 수 있다 (fractional indexing)
- [x] **CHAP-06**: 챕터에 상태를 지정할 수 있다 (초안/검토 중/완성)

### WORLD — 세계관 데이터베이스

- [x] **WORLD-01**: 사용자가 캐릭터를 생성·수정·삭제할 수 있다
- [x] **WORLD-02**: 사용자가 장소를 생성·수정·삭제할 수 있다
- [x] **WORLD-03**: 사용자가 세계관 설정 항목을 생성·수정·삭제할 수 있다
- [x] **WORLD-04**: 사용자가 시간표/연표 항목을 생성·수정·삭제할 수 있다
- [x] **WORLD-05**: 사용자가 캐릭터 간 인간관계를 정의할 수 있다
- [x] **WORLD-06**: 세계관 항목 목록에서 검색 및 필터링이 가능하다

### EDITOR — 사이드패널 컨텍스트

- [x] **EDIT-01**: 챕터 에디터는 좌측 에디터 + 우측 사이드패널의 2-패널 레이아웃으로 표시된다
- [x] **EDIT-02**: 사이드패널에 현재 챕터에 관련된 캐릭터/장소/세계관 항목이 목록으로 표시된다
- [x] **EDIT-03**: 사용자가 사이드패널에서 개별 항목을 "AI 컨텍스트에 포함" 토글로 선택/제외할 수 있다
- [x] **EDIT-04**: 선택된 컨텍스트 항목의 요약이 AI 생성 요청 시 시스템 프롬프트에 자동으로 포함된다

### AI — AI 초안 생성

- [x] **AI-01**: 사용자가 챕터 에디터에서 "AI 초안 생성" 버튼을 클릭하면 AI가 챕터 초안을 작성한다
- [x] **AI-02**: AI 생성 결과가 SSE(Server-Sent Events)로 에디터에 실시간 스트리밍된다
- [x] **AI-03**: AI 생성 중에는 에디터가 read-only 상태로 전환되고 취소 버튼이 표시된다
- [ ] **AI-04**: 사용자가 AI 모델을 선택할 수 있다 (Claude Sonnet/GPT-4o 등) — **DEFERRED to v2**
- [ ] **AI-05**: AI 생성 시 이전 챕터 요약이 컨텍스트에 자동으로 포함된다 — **DEFERRED to v2**
